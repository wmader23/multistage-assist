# Timers

Set countdown timers with optional device notifications.

## Commands

| Command | Result |
|---------|--------|
| "Timer f端r 10 Minuten" | 10-minute timer |
| "Timer 30 Sekunden" | 30-second timer |
| "Timer f端r 5 Minuten Eier" | Named timer "Eier" |
| "Pizza Timer f端r 15 Minuten" | Named timer "Pizza" |

## Duration Formats

- X Minuten / X Minute
- X Sekunden / X Sekunde  
- X Stunden / X Stunde

## Device Notifications

Specify which device to notify:

```
"Timer f端r 10 Minuten auf Daniels Handy"
```

The system can learn device aliases for future use.

## Keywords

Words that trigger timer mode:
- timer
- wecker
- countdown
- stoppuhr

Note: "Uhr" is NOT included (conflicts with time expressions like "15:00 Uhr").
